# Approaches - Iterated Prisoner's Dilemma

All strategy implementations for APOGEE'24 & APOGEE'25 IPD challenge.

## Quick Start

```bash
python3 eval_engine.py
```

Edit `eval_engine.py` to change the matchup (lines ~25-30).
